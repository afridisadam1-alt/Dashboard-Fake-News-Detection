# Fake News Detection Project

## Overview
This project trains and evaluates deep learning and machine learning models for fake news detection on multiple datasets (FA-KES, George McIntire, ISOT, EUvsIGF, EUvsDisinfo, ISOT, EUvsISOT).

## Requirements
- streamlit==1.50.0
- pandas==2.3.3
- numpy==1.26.4
- PyPDF2==3.0.1
- python-docx==0.8.11
- tensorflow==2.16.2
- altair==5.5.0
- wordcloud==1.9.2
- gdown==5.2.0
- matplotlib==3.8.1
- openpyxl==3.1.3
- xlrd==2.0.1
scikit-learn==1.6.1
## Setup
1. Clone the repository.
2. Install required packages:
3. Python3 MLtrain_models.py
4. Python3 DLtrain_models.py
5. streamlit run app.py
