# Fake News Detection Project

## Overview
This project trains and evaluates deep learning and machine learning models for fake news detection on multiple datasets (FA-KES, Pandy, EUvsIPF, EUvsDisinfo, ISOT, EUvsISOT).

## Requirements
- Python 3.9+
- Packages:
  - numpy
  - pandas
  - scikit-learn
  - tensorflow
  - matplotlib
  - nltk
